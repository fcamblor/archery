# Towerfall PhaserJS

Jeu multijoueur en réseau inspiré de Towerfall, développé avec Phaser 3.

## Stack technique

- **Moteur** : Phaser 3 (arcade physics)
- **Langage** : TypeScript
- **Bundler** : Vite
- **Commandes** : `npm run dev` (dev server), `npm run build` (production)

## Conventions

- `plans/` et `docs/` contiennent le plan d'itérations, le game design et la documentation technique — les maintenir à jour au fil des itérations.
- **À chaque fin d'itération** : mettre à jour la documentation (`plans/`, `docs/`) pour qu'elle reste synchronisée avec le code.
- **Messages de commit en français** : tous les messages de commit doivent être rédigés en français.
